# Надёжность источников
- **Tier 1:** Официальные доки/спеки/релизы, peer-reviewed исследования, первичные данные (фин. отчёты, гос данные), прямые интервью/тесты.
- **Tier 2:** Авторитетные медиа с фактчеком, аналитики, отраслевые издания с редакцией.
- **Tier 3:** Блоги/форумы/соцсети/анонимные источники — использовать только с подтверждением Tier1/2.
