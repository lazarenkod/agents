# Writing Checklist
- Аудитория/цель/канал определены
- Структура (SCQA/Problem-Solution/How-to) выбрана и соблюдена
- Шаги/примеры/код проверены, версии указаны
- Термины согласованы, глоссарий есть
- Факты/ссылки/PII/легал проверены
- Тон/бренд соблюдены, CTA есть
- Форматирование/визуалы читаемы
- Орфография/грамматика/пунктуация исправлены
