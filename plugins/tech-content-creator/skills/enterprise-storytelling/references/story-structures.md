# Структуры сюжетов для тех-контента
- **Hero’s Journey (упрощённо):** герой → вызов → наставник → испытания → победа → возврат с наградой.
- **Problem–Agitate–Solve:** проблема → усугубление → решение → выгоды.
- **Before/After/Bridge:** было → стало → как добились.
- **Myth/Reality:** распространённое мнение → факты/данные → что делать.
- **Case Spine:** контекст → конфликт → решение → результат → уроки → CTA.
