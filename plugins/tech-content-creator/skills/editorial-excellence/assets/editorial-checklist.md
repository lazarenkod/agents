# Editorial Checklist

- Цель/аудитория/канал ясны
- Заголовок/лид цепляют, обещание выполняется
- Структура логична, нет лишних секций, переходы есть
- Ясность/сжатие: жаргон минимизирован, примеры/цифры добавлены
- Факты/источники/цитаты проверены, ссылки указаны
- Терминология единообразна, стиль/тон = бренд-гайд
- CTA понятен, следующий шаг прозрачен
- Форматирование/визуалы помогают, не мешают
- Орфография/грамматика/пунктуация исправлены
- Легал/бренд/локализация проверены
