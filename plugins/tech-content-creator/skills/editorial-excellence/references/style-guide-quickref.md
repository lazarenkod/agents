# Style Guide Quick Reference
- Тон: {формальный/дружелюбный/партнёрский}, избегать {клише/слэнг/агрессия}.
- Предпочитаемые термины/написания (бренд/продукты/единицы измерения).
- Пунктуация и регистр (заголовки, списки, аббревиатуры).
- Числа и валюты: формат, округление, локаль.
- Ссылки: формат, атрибуция, проверки.
