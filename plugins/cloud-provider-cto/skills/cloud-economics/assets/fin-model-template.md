# Финмодель (cloud economics)

## Исходные данные
- Сервисы/продукты, цены, скидки (RI/SP/EA), egress, support
- Нагрузка/прогноз, регионы/PII, инфраструктура/CapEx (если есть)

## Сценарии
- Base / Upside / Downside
- Метрики: $/unit, COGS, маржа, payback, runway, egress %

## Сенситивити
- Драйверы: нагрузка, цены, egress, скидки, утилизация
- Таблица чувствительности и выводы

## План действий
- Топ-инициативы, экономия, сроки, риски, владельцы
