# Playbook оптимизаций
- Rightsizing/Autoscaling: профилирование, лимиты, авто-скейл правила.
- Pricing: RI/SP/EA, резервирование емкости, spot для batch, Savings Plans.
- Storage: классы (hot/warm/cold), lifecycles, dedupe/compaction, архив, cleanup.
- Network: egress/ingress оптимизация, локализация данных, CDN/peering.
- Architecture: multi-AZ/region выбор, кеши, компактация запросов, RPS smoothing.
- Governance: теги, бюджеты/квоты, алерты, chargeback/showback, политики лабораторий.
