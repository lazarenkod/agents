# Контрактный чек-лист (облако)
- Скидки/коммиты (RI/SP/EA), срок действия, штрафы/rollover.
- Egress/сетевые сборы, support план, SLA и кредиты.
- Лицензии/PII/региональность данных, аудиты/комплаенс.
- Правила роста/расширения, возможность renegotiate.
- План обновления/экспирации, напоминания о дедлайнах.
