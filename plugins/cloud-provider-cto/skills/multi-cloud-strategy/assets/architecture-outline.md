# Multi-Cloud Architecture Outline

## Network
- Connectivity (VPN/Direct/Interconnect), DNS/GSLB, segmentation, egress

## Identity/Security
- IAM/SSO/PKI унификация, secrets, policy-as-code, logging/siem

## Data
- Репликация/согласованность, классификация/PII/суверенитет, форматы/ETL, кеш/edge

## Observability
- Метрики/logs/traces, единая панель, алерты, дрейф-конфиг

## CI/CD и операции
- Артефакты/репозитории, фич-флаги, канареек/blue-green, DR/failover тесты

## Governance/Стоимость
- Теги/квоты/бюджеты, egress/стоимость дублирования, контракты/скидки
