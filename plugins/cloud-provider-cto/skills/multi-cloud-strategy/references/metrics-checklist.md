# Чек-лист метрик multi-cloud
- Доступность/SLO, latency p50/p95/p99, успех DR тестов, время failover.
- Стоимость: egress, дублирование данных, ops/tooling, RI/скидки.
- Данные: задержки репликаций, рассинхрон, ошибки консистентности.
- IAM/безопасность: дрейф политик, открытые ресурсы, ключи/сертификаты.
- Операции: MTTR при фейлах, дрейф конфигов, количество ручных шагов.
