# Риски multi-cloud
- Стоимость: egress, дублирование данных/инструментов, ops нагрузка.
- Сложность: IAM/PKI, сети, observability, CI/CD, поддержку.
- Данные: суверенитет/PII, согласованность, сплит-brain.
- Надёжность: несогласованные процедуры DR, редкие тесты, дрейф конфигов.
- Команда: нехватка экспертизы, разные SDK/CLI, разные модели биллинга/контрактов.
