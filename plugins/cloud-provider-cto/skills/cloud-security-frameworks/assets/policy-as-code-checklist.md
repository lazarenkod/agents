# Policy-as-Code Checklist

- Базовый набор контролей (IAM/Network/Data/Secrets/Logging/SLO/SIEM).
- Шифрование at rest/in transit, ключи/ротация, KMS/Secrets Manager.
- Public access блоки: SG/ACL, buckets/obj storage, LB/ingress.
- Compliance guardrails (CIS/NIST/ISO/PCI/HIPAA/GDPR) и дрифт-детект.
- Алёрты/каналы/владельцы, процесс исключений, тесты/валидация.
