# Чек-лист метрик безопасности
- Coverage: теги, политики, IaC, шифрование, MFA, ротация ключей.
- Инциденты: MTTR, частота, критичность, RCA закрытие.
- Дрейф/уязвимости: # отклонений от baseline, время до фикса, патчи/сканы.
- Access: открытые ресурсы, публичные buckets/SG, stale accounts/keys.
- Compliance: % пройденных контролей, аудитов, исключения/waivers.
