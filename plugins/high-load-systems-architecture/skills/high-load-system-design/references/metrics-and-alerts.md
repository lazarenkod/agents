# Метрики и алерты
- Latency p50/p95/p99, error rate, saturation (CPU/mem/disk/net), queue depth, cache hit, tail latency.
- Capacity: headroom, autoscale events, backlog, GC/pauses, thread pools.
- Бизнес/стоимость: cost per RPS/req, SLA кредиты, квоты.
- Алерты: рост tail latency/ошибок, resource exhaustion, конфиг drift, SLA breach, деградация cache/queues.
