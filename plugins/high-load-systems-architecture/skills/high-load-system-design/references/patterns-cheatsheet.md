# Patterns Cheatsheet (High Load)
- CQRS/Event sourcing, Saga (orchestration/choreography), API Gateway/BFF.
- Caching: read-through/write-through/write-behind, cache-aside, TTL/eviction.
- Queues/Streams: work queues, pub/sub, backpressure, replay/ordering.
- Partitioning: range/hash/geo/directory, hot-spot mitigation, rebalancing.
- Resilience: circuit breakers, bulkheads, retries+backoff+idempotency, rate limiting.
