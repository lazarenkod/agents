# SLO/SLI Guide
- Определи SLI: latency, availability, error rate, freshness, correctness (если применимо).
- Установи SLO: p50/p95/p99 цели, error budget, окна.
- Мониторинг: вычисление SLI, burn rate алерты (1h/6h/24h), каналы/владельцы.
- Управление: policy по использованию error budget (freeze/review), регулярные обзоры.
