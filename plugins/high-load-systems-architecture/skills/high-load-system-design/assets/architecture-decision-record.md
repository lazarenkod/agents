# Architecture Decision Record (ADR)

## Контекст
- Цель/проблема, SLO/нагрузка, ограничения

## Варианты
- Option A (плюсы/минусы/риски)
- Option B (...)
- Option C (...)

## Решение
- Выбранный вариант, обоснование (trade-offs, цифры)

## План
- Шаги/фазы, владельцы/сроки, миграции/rollback

## Метрики/алерты
- SLO/SLA, ключевые метрики, пороги стопа

## Последствия
- Техдолг/стоимость, риски, что пересмотреть позже
