# План кросс-регионального фейловера Bedrock
- Актив/пассива регионы, списки моделей/эндпоинтов, зависимости (KB, S3, AOSS).
- Репликация: KB данные, конфиги, секреты (SSM/Secrets Manager), CloudWatch dashboards.
- Переключение: DNS/Route53 health checks, feature flag для маршрутов, downgrade max_tokens.
- Тесты: квартальные DR-учения, RTO/RPO измерения, проверка guardrails в целевом регионе.
- Риски: ценовые различия, квоты в резервном регионе, несовпадение версий моделей.
