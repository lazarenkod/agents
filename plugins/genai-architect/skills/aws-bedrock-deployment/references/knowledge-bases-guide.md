# Bedrock Knowledge Bases guide
- Источники: S3 + AOSS/OpenSearch Serverless; форматы (PDF/MD/HTML); PII/retention.
- Embeddings: Titan embed, сторонние; размерность, метаданные, фильтры.
- Индексация: пайплайн ingestion, дедуп, chunking, версионирование, lag алерты.
- Запросы: hybrid search (BM25+dense), rerank, цитаты; guardrails на вывод.
- Ops: мониторинг latency/recall, KB health, replication/multi-AZ, backup/restore, DR тесты.
