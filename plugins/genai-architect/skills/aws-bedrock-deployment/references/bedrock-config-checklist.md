# Bedrock Config Checklist
- VPC + privatelink, endpoints, security groups, KMS
- IAM: least privilege, per-model permissions, logging/audit
- Provisioned Throughput sizing, autoscale, quotas/alerts
- Context/tokens limits, cache/batch, routing, Guardrails
- Logging/observability: metrics/logs/traces, SIEM, retention
- Safety/PII: moderation, redaction, storage/log policies
