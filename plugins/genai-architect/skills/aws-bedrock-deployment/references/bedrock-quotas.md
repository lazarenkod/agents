# Квоты и лимиты Bedrock
- Tokens: max per запрос/день по модели; повышение через Support (use-case, трафик).
- Provisioned Throughput: max MU per модель/регион, шаг увеличения, сроки активации.
- Concurrency: invoke limits, streaming соединения, KB ingestion rate.
- Storage: AOSS collection limits, индексы, размеры.
- Мониторинг: алерты на throttling, utilisation PTU, KB lag.
