# Шаблоны политик
- Acceptable Use Policy (AUP) для пользователей/тенантов.
- Доступ к PII: маскирование, хранение, ретеншн, экспорт.
- Генерация контента: допустимые/запрещённые темы, дисклеймеры.
- Инциденты: уведомления, эскалации, RTO/RPO для данных.
- Версионирование и approvals: владельцы, ревью cadence, трассируемость.
