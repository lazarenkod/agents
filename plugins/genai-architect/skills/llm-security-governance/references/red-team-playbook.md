# Red-team playbook для LLM
- Цели: jailbreak, prompt injection, data exfil, policy bypass, toxicity.
- Инструменты: списки атак (семплы), автоматизация (LLM-as-attacker), fuzzing контекста.
- Процесс: план → запуск → сбор метрик (pass/fail, severity) → фиксы → ретест.
- Отчёт: кейсы, воспроизведение, корневые причины, контроли/патчи, SLA по исправлениям.
- Частота: перед релизом, ежеквартально; включает многоязычные атаки и мультимодальные, если релевантно.
