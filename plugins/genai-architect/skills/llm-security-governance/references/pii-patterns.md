# PII patterns (шпаргалка)
- Имена/адреса, email/телефон, паспорта/SSN/ИНН, карты, аккаунты, API keys/секреты.
- Регулярки/модели: Presidio/Detoxify/custom regex; языки/локализация.
- Политики: redaction/анонимизация/блокировка, логирование с масками, retention.
- Тесты: выборка синтетических/реальных кейсов, метрики precision/recall, false positives.
