# OWASP LLM Top 10 (кратко)
- Prompt Injection, Insecure Output Handling, Training Data Poisoning, Model Denial of Service, Supply Chain, Sensitive Info Disclosure, Excessive Agency, Overreliance, Model Theft, Monitoring/Logging Failures.

- Использование: чеклист при дизайне/ревью, связь с guardrails и тест-планом.
