# Checklist: оценка политик/guardrails
- Категории: PII, безопасность, юридические/региональные ограничения, бренд/тон.
- Тест-кейсы: блок/редактирование, allow/deny списки, safe completion fallback.
- Покрытие: языки, форматы (текст/картинки), роли пользователей, контекстные атаки.
- Метрики: pass rate, false positives, время на модерацию, дрейф качества.
- Документация: версии политик, владельцы, дата последнего ревью.
