# Guardrails Implementation Checklist

- Policy/safety фильтры до/после, self-critique, block/allow lists
- PII: детекция/маскирование/ретеншн, безопасные логи/кеши/хранилища
- Prompt injection: sanitization, input filters, контекст валидация
- Output: grounding/citations, refusal patterns, toxicity/bias checks
- IAM/доступ: принципы least privilege, audit logs, segregation of duties
- Red teaming/пентест: сценарии, частота, remediation план
- Алерты: policy/PII violations, дрейф моделей/данных, превышения квот
