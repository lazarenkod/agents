# Runbook: обработка PII
- Триггеры: обнаружение PII в вводе/выводе, алерты политики, инциденты клиентов.
- Немедленно: блок/редактирование ответа, логирование в audit, уведомление on-call + privacy.
- Диагностика: источник PII (ввод/кеш/база), маршруты, модели.
- Устранение: обновить правила детекта/редакции, очистить кеш/логи, усилить фильтры.
- Пост-мортем: обновить документы, обучение, пересмотр порогов/покрытия.
