# Калькуляторы стоимости
- Провайдерские: OpenAI/Azure, AWS Bedrock, Vertex AI — встроенные калькуляторы.
- Самостоятельные: spreadsheet с токен-профилями, сценарии base/upside/downside, $/req, $/user, бюджет/лимит.
- Инфраструктура: GPU/CPU час, storage, egress; амортизация self-host.
- Практика: сверять фактические метрики (billing) с прогнозами, алерты на расхождение.
