# Техники оптимизации GenAI
- Prompt/контекст: сжатие, шаблоны, инструкции, ограничение длины, резюме.
- Кеширование: semantic/LLM output cache, partial caching, eviction/TTL.
- Батчинг: группировка запросов, токен-алайнмент, async pipelines.
- Маршрутизация: tiered модели, классификация сложности, функции vs генерация.
- Distillation/тонкие модели/quantization для self-host.
- Retrieval/filtering перед моделью; stop-early/self-check.
- Scheduling: оффлайновые задачи/embeddings в low-cost окна.
