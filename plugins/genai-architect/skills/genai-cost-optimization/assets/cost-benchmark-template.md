# Шаблон cost-бенчмарка
- Цель: сравнить $/req, latency, качество между моделями/роутами.
- Нагрузка: типы запросов (simple/complex), длина, объём выборки.
- Метрики: $/1k токенов, $/req, p95 latency, quality (eval), кеш hit.
- Формат: таблица результатов, графики latency/cost, вывод/рекомендация.
- Частота: при релизах моделей, ежемесячный пересмотр.
