# Платформенные бенчмарки
- Метрики: p95/p99 latency, tokens/s, cost/1k токенов, availability, throttle rate.
- Нагрузки: короткие/длинные контексты, streaming vs non-streaming, batch sizes.
- Провайдеры: Bedrock, Azure OpenAI, Vertex AI, self-host Triton/NIM.
- Формат отчёта: таблица результатов, графики latency/cost, рекомендации выбора.
- Частота: ежемесячно, при релизах моделей/конфигов.
