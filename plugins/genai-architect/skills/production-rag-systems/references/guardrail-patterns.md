# Паттерны guardrails
- Встроенные фильтры: классификация prompt/response (toxicity, self-harm, IP).
- PII: детект + редакция, санкционированные маски, audit log.
- Groundedness: cite threshold, abstain при низком overlap, verifier model.
- Policy: deny/allow списки тем, региональные ограничения, юридические дисклеймеры.
- Тесты: red-team сценарии, jailbreaks, prompt injection, data exfiltration.
