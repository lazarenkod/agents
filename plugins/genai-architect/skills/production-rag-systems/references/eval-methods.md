# Методы оценки RAG
- Offline: golden datasets, LLM-as-judge (pairwise/scale), human review, regression suites.
- Online: A/B/канарейки, интерливинг retrieval, метрики uplift, guardrails.
- Safety: adversarial prompts, prompt injection, policy/bias, PII leakage tests.
- Data drift: мониторинг качества индекса/ответов при обновлениях, дрейф embedding.
