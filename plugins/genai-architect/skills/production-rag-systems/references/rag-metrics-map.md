# Карта метрик RAG
- Качество: groundedness/faithfulness, цитаты, retrieval precision/recall, rerank NDCG/MRR.
- Производительность: latency p50/p95/p99 (retrieval/rerank/generation), throughput/qps, cache hit rate.
- Стоимость: $/ответ, $/ингест, storage, egress.
- Безопасность: policy/PII violations, отказ/фильтр rate, self-critique success.
- Надёжность: error rate, timeout, refresh/index drift, тесты регрессии pass rate.
