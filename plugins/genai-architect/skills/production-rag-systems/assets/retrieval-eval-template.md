# Шаблон оценки retrieval
- Цель: точность поиска, recall, drift.
- Датасет: золотые пары вопрос/ответ/док, размер, обновление.
- Метрики: Recall@k, MRR, nDCG, coverage, latency p95.
- Эксперименты: BM25 vs dense vs hybrid, rerankers, top-k/top-payload.
- Отчёт: таблица результатов, выводы, выбранная конфигурация, риски.
