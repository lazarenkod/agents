# Библиотека промтов (шаблон)
- System prompt: цель, тон, цитаты/ссылки, безопасные инструкции.
- Retrieval prompt: формат контекста, max tokens, порядок (новый→старый), инструкции на русский.
- Answer prompt: стиль, краткость, cite формат, отказ при отсутствии фактов.
- Troubleshooting: fallback prompt при пустом контексте, guardrails для PII/policy.
- Версионирование: id/дата/владелец, A/B эксперименты, дифф изменений.
