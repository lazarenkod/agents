# Фреймворки продуктовой аналитики

- **North Star + Input Metrics:** связывает основную ценность с драйверами (активация, вовлечённость, удержание, монетизация).
- **AARRR:** Acquisition, Activation, Retention, Referral, Revenue — общая воронка.
- **HEART:** Happiness, Engagement, Adoption, Retention, Task Success — для UX/фич.
- **Growth Accounting:** New + Resurrected – Churn; Quick Ratio для здоровья роста.
- **LTV/CAC & Payback:** экономика роста и эффективность каналов.
- **Experiment Lifecycle:** Hypothesis → Power → Design → Guardrails → Decision → Rollout → Learnings.
- **Data Quality Gates:** свежесть данных, полнота, дубликаты, согласованность схем, тесты на дрейф.
