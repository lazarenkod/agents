# Шпаргалка по фреймворкам

- **CIRCLES (Product Sense):** Comprehend → Identify → Report → Cut → List → Evaluate → Summarize. Всегда фиксируй пользователя, цель, метрики, trade-offs.
- **GAME (Execution/Analytics):** Goals → Actions → Metrics → Evaluation. Добавляй guardrails и риск-метрики.
- **STAR (Behavioral):** Situation → Task → Action → Result → Learning. Коротко в S/T, подробно в A/R.
- **System Design Steps:** Требования → Оценки → API → Архитектура → Глубокие dive → Trade-offs. Пиши метрики SLO, объёмы, кэш/шардинг.
- **PRFAQ/Working Backwards:** полезно как форма Product Sense ответа.
- **Metrics Toolkit:** NSM + AARRR, HEART, LTV/CAC, выпуск критерии.
