# Платформенные фреймворки

- **Flywheel:** предложение → разработчики → приложения → ценность → спрос → предложение.
- **Two-Sided Network Effects:** ценообразование/субсидии для сторон, баланс качества предложения/спроса.
- **Platform Triangle:** Value (что дают API/SDK), Control (governance, качество), Monetization (pricing, take rate).
- **API Lifecycle:** дизайн → версия → тест → выпуск → депрекейшн; строгие контракты и миграции.
- **Marketplace Governance:** допуск, ревью, рейтинги, фрод-контроль, спор/рефанд политика.
- **Data Network Effects:** больше данных → лучше качество → больше пользователей → ещё данные (важны privacy/PII границы).
