# DX Checklist (разработчики)

- [ ] Onboarding <5 минут: регистрация, ключ, первый запрос
- [ ] Песочница и готовые коллекции (Postman/curl)
- [ ] SDKs для основных языков/платформ
- [ ] Примеры end-to-end (web/mobile/backend)
- [ ] Чёткая политика версий и депрекейшна
- [ ] Rate limits, quotas, идемпотентность, retriable ошибки
- [ ] Webhook подписи и переотправка
- [ ] Логирование/трассировка для разработчиков
- [ ] SLA/поддержка: каналы, время ответа, статус-пейдж
- [ ] Changelog и roadmap для партнёров
