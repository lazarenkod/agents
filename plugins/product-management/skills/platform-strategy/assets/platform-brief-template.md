# Платформенный бриф

## 1. Контекст и цель
- Для кого платформа (producers/consumers/partners)
- Job-to-be-done и ценность по сторонам
- NSM и главные драйверы

## 2. Flywheel и модель
- Описание flywheel, текущие слабые звенья
- Тип модели: marketplace / usage API / extensibility
- Ключевые стимулы (субсидии, рев-шеры, кредиты)

## 3. Экономика и монетизация
- Pricing/packaging: freemium/usage-tiered, rev-share
- Take rate, attach rate, payback партнёров
- Сценарии роста/сезонность/сенситивити

## 4. DX и качество
- Time-to-first-call, примеры и SDK
- SLO/SLA, документация, обратная связь
- Governance: ревью приложений, безопасность, политика данных

## 5. Риски и гарнды
- Технические, правовые, фрод, зависимость от сторонних API
- План мониторинга и алертов
- Rollback/fallback сценарии

## 6. План и метрики
- Этапы: pilot → beta → GA
- KPI: активные разработчики, успешные интеграции, GMV/take rate, NPS/DX
- Стейкхолдеры и владельцы задач
