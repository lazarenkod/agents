# Чек-лист облака для аттестации ФСТЭК
- Гипервизор/виртуализация: доверенные версии, изоляция тенантов, защита от side-channel.
- Сеть: сегментация, VPN/IPsec/ГОСТ, межсетевые экраны, DDoS защита, контроль маршрутов.
- Хранилища: шифрование, снапшоты/backup изолированы, контроль доступа и журналирование.
- Администрирование: MFA, RBAC/ABAC, раздельные контуры, журналирование действий, change management.
- СЗИ: перечень сертифицированных средств (СОВ/СЗИ НСД/СОВ/МЭ), актуальные сертификаты.
- Логи/мониторинг: централизованный сбор, защита/ретеншн, SIEM, синхронизация времени.
- Документы: модель угроз, матрица контролей, программа/методика испытаний, evidence pack, план устранения.
