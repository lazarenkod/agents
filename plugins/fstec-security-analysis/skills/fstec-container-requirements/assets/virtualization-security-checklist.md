# Чек-лист безопасности виртуализации
- Гипервизор: доверенная версия, Secure Boot/TPM, отключение kexec и ненужных модулей/сервисов.
- Управление: MFA, RBAC/ABAC, аудит админских действий, change management, разделение доменов администрирования.
- Изоляция: сегментация mgmt/tenant/storage, контроль L2/L3, защита от side-channel (SMT policy, CPU pinning, LLC partitioning если доступно).
- Миграции: шифрованные каналы live-migration, запрет межконтурных миграций без аттестации, проверка целостности.
- Хранилища/образы: шифрование VM/backup, контроль снапшотов/клонов, подпись образов, ограниченный доступ к datastore.
- Сеть: VPN/IPsec/ГОСТ для управления, firewall, egress control, доверенный DNS/NTP, DDoS защита.
- Логи/мониторинг: централизованный сбор, защита/ретеншн, алерты на критические действия, синхронизация времени.
- СЗИ/сертификация: перечень сертифицированных средств (СОВ/СЗИ НСД/МЭ), актуальные сертификаты, совместимость с RedOS 7/8.
