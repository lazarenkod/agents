# Evidence Map (контейнеризация)

- **Доступ/усиление:** RBAC политики, списки пользователей/ключей, MFA, audit logs.
- **Сеть:** NP/firewall конфиги, TLS/mTLS настройки, egress/ingress правила, журналы сетевых событий.
- **Данные:** шифрование (at rest/in transit), KMS политики, бэкап/восстановление отчёты, контроль снапшотов.
- **Образы/CI/CD:** отчёты сканов, подписи, политики допуска, SBOM, секреты/хранилища, логи сборок/деплоев.
- **Логи/мониторинг:** конфигурация агрегаторов, ретеншн/целостность, синхронизация времени, SIEM/SOAR правила.
- **Уязвимости/патчи:** отчёты по CVE, SLA трекер, дрифт IaC/конфигов.
- **IR/BCP:** планы, отчёты тестов, постмортемы, обучение.
