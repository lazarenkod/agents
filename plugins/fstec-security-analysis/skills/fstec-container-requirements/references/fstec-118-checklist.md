# Чек-лист 118 для контейнеров
- Организационные меры: политика, роли, журналы, обучение.
- Доступ: IAM/RBAC, аутентификация, ключи/сертификаты, MFA, сегрегация ролей.
- Сеть: сегментация/NP, firewall, шифрование каналов, контроль внешних подключений.
- Данные: шифрование at rest/in transit, контроль доступа, резерв/архив, целостность.
- Логирование/мониторинг: централизованный сбор, защита логов, время/синхронизация.
- Уязвимости/обновления: сканы образов/узлов, патчи, подписи/целостность, supply chain.
- CI/CD/registry: подпись, политика образов, доступы, изоляция сборок, хранение секретов.
- Реагирование: планы, алерты, тесты, обучение, журналы действий.
