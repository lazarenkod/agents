# Требования ФСТЭК к виртуализации (ориентиры)
- Нормативка: приказы 17/21, методики ФСТЭК по защите виртуализации; учитывать 118 при контейнерах внутри VM.
- Гипервизор/хосты: доверенные версии, Secure Boot/TPM, контроль загрузки/модулей, отключение kexec, регулярные патчи.
- Изоляция: сегментация mgmt/tenant/storage, защита от межтенантных атак (L2/L3/side-channel), политика SMT/NUMA, ограничение shared ресурсов.
- Управление доступом: MFA, RBAC/ABAC, раздельные учётки, журналирование админов, ревью привилегий.
- Сеть/каналы: шифрование управления/миграций (VPN/IPsec/ГОСТ), firewall, egress control, доверные DNS/NTP, защита от DDoS.
- Хранилища/образы: шифрование VM и backup, контроль снапшотов, подпись образов, политика уничтожения, доступ к datastore по принципу НОП.
- Миграции/HA: только между доверенными хостами/контуром, шифрованные каналы, контроль целостности, запрет миграций между неаттестованными зонами.
- Логи/мониторинг: централизованный сбор с гипервизоров/управляющих, SIEM, защита/ретеншн логов, time sync.
- СЗИ/сертификация: перечень сертифицированных средств (СОВ/СЗИ НСД/МЭ), актуальные сертификаты, совместимость с RedOS/выбранной ОС.
