# Контроли по компонентам контейнеризации
- **Control Plane:** RBAC/ABAC, audit, TLS/mTLS, API auth, ограничение kubeconfig, защита ETCD, бэкапы/шифрование.
- **Worker:** hardening ОС, ограничение привилегий, sandbox/secure runtimes, selinux/AppArmor, patching/EDR.
- **Registry:** подпись/скан, authz, приватность, ретеншн, репликации, мониторинг push/pull.
- **CI/CD:** защита секретов, изоляция пайплайнов, SBOM, политика зависимостей, MFA, аудиты.
- **Network:** сегментация, NP, ingress/egress, firewall, DDoS защита, DNSSEC, time sync.
