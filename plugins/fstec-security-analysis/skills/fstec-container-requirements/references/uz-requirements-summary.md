# Требования по уровням защищенности (УЗ) — кратко
- **УЗ-1:** максимальные требования, мандатная модель, обязательное шифрование/аудит, сертификация всех компонентов.
- **УЗ-2:** строгие меры, расширенные контроли доступа/логирования, защита сети/данных, сертификация.
- **УЗ-3:** стандартные меры, контроль доступа, шифрование, журналы, обновления/патчи, контроль целостности.
- **УЗ-4:** базовые меры, минимальные контроли для ограниченного доступа.
