# Требования ФСТЭК к облачным средам
- Изоляция: сегментация по тенантам, доверенный гипервизор, защита от межтенантных атак (L2/L3/side-channel).
- Доступ и администрирование: MFA, RBAC/ABAC, раздельные контуры управления, журналирование действий админов, контроль изменений.
- Сеть/каналы: VPN/IPsec/ГОСТ, межсетевые экраны, фильтрация egress, защита от DDoS, мониторинг маршрутизации.
- Хранение/бэкапы: шифрование at rest, снапшоты/backup с изоляцией, контроль доступа, ретеншн, целостность.
- СЗИ/сертификация: перечень сертифицированных средств (СОВ/СЗИ НСД/СОВ/МЭ), актуальные сертификаты, поддерживаемые версии ОС (RedOS 7/8 и др.).
- Аттестация: модель угроз, матрица контролей, программа/методика испытаний, evidence pack, управление изменениями после аттестации.
