# Чек-лист готовности облака к сертификации ФСТЭК
- Модель обслуживания: SaaS/PaaS/IaaS, разделение ответственности, границы тенантов.
- Гипервизор/виртуализация: доверенная версия, изоляция VM/контейнеров, контроль side-channel.
- Сеть: сегментация, VPC/VNet, межсетевые экраны, VPN/IPsec/GOST, DDoS/SOC.
- Хранилища: шифрование (СКЗИ), снапшоты/backup, контроль доступа/журналирование.
- Управление доступом: RBAC/ABAC, MFA, раздельные контуры администрирования, jump-host.
- Журналы/мониторинг: полный аудит (доступ/конфигурации/СЗИ), SIEM/SOAR, синхронизация времени.
- СЗИ/сертификация: перечень сертифицированных средств (СОВ, СЗИ НСД/СВТ/СОВ), актуальные сертификаты.
- Аттестация: программа/методика испытаний, матрица контролей, план устранения несоответствий, evidence pack.
