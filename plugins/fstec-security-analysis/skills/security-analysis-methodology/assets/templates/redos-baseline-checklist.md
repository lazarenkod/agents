# Чек-лист RedOS 7/8 (сертифицированные узлы)
- Репозитории: только доверенные, контроль хэшей/подписей, запрет сторонних пакетов.
- Ядро/модули: версия по сертификации, отключить ненужные модули, auditd включен.
- SELinux: enforcing, собственные policy для kubelet/containerd/docker.
- Сеть: firewalld/nftables правила, отключение IPv6 если не требуется, time sync, sysctl hardening.
- Криптография: сертифицированные СКЗИ, TLS профили (GOST где нужно), KMS/ХКИ.
- Логи/аудит: централизованный сбор, запрет локального удаления, ретеншн/целостность.
- Учётные записи: MFA/парольная политика, sudo без паролей запрещён, блок неиспользуемых.
- Обновления: регламент/окна, тестовый контур, rollback plan, журнал изменений.
