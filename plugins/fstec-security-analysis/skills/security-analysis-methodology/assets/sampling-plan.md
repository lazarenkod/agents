# План сэмплирования

## Цель
- На что распространяется выборка (образы/узлы/CI-CD/registry/NP/RBAC)

## Критерии отбора
- Критичность (prod/PII/КИИ), версии/возраст, частота изменений, инциденты/жалобы

## Объём выборки
- N объектов на категорию, период/окно времени, рандом/целенаправленно

## Процедуры
- Методы проверки (скан/ревью/тест), артефакты, пороги pass/fail

## Отчёт
- Покрытие, найденные несоответствия, выводы, задачи
