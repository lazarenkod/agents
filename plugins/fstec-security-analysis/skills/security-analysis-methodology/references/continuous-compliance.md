# Continuous Compliance (118)
- Автоматизация: policy-as-code, непрерывные сканы образов/узлов, drift detection IaC, алерты на нарушения.
- Частота: ежедневные/еженедельные проверки критичных контролей, ежемесячные обзоры, квартальные аудиты.
- Ритм: WBR/MBR по несоответствиям, SLA на remediation, авто-тикеты.
- Документы: актуализация модели угроз, обновление матрицы контролей и evidence pack.
- Закрепление: KPI/OKR по закрытию несоответствий, обязательный статусный отчёт, периодическая переоценка рисков.
