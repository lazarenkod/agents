# Reporting Templates (ориентиры)
- Финальный отчёт: резюме, scope, методология, результаты, риски, рекомендации, приложения.
- Менеджмент-резюме: топ-риски/несоответствия, план мер, сроки/владельцы, зависимые контуры.
- Статусные апдейты: прогресс по контролям/мероприятиям, просрочки, блокеры, алерты.
