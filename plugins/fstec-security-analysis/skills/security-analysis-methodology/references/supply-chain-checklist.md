# Supply Chain Checklist (контейнеры/CI-CD)
- Подпись артефактов, проверка подписи на deploy.
- Скан зависимостей/образов (CVE/SBOM), политика блокировки критичных CVE.
- Изоляция сборок, контроль секретов, MFA для CI/CD, аудиты изменений.
- Registry: доступ по ролям, репликации, защита от публичного pull/push, ретеншн/очистка.
- Мониторинг и алерты на аномалии (необычные push, скачки трафика, новые образы без подписи).
