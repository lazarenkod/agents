# Руководство по оценке рисков
- Шкала impact (финансовый/регуляторный/операционный/репутационный), уровни 1–5.
- Шкала вероятности (редко → часто), критерии для каждой категории.
- Матрица риск = impact × probability; пороги для крит/высок/сред/низ.
- Связка с remediation: сроки, владельцы, алерты, необходимость обоснования исключений.
