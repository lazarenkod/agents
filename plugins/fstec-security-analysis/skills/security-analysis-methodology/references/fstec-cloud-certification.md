# Требования ФСТЭК к сертификации облачных сред
- Нормативка: приказы 17/21, 118 (если контейнеры), методические материалы по аттестации облаков, требования к СКЗИ.
- Изоляция: многоуровневая сегментация тенантов, гипервизор с контролем НСД, защита от межтенантных атак (side-channel, L2/L3).
- Администрирование: раздельные контуры управления, контроль привилегий, журналирование действий админов, двухфакторная аутентификация.
- Сеть/каналы: обязательное шифрование, VPN/IPsec/ГОСТ, защита от DDoS, контроль маршрутизации/доверенных зон.
- Хранилище/бэкапы: шифрование at rest, управление ключами, изоляция снапшотов, контроль целостности, ретеншн.
- Мониторинг/аудит: централизованный сбор логов (доступ, конфиг, СЗИ), SIEM, синхронизация времени, хранение и защита журналов.
- СЗИ: перечень сертифицированных средств (СОВ/СЗИ НСД/СОВ/МЭ), актуальные сертификаты, поддержка в целевых версиях RedOS/других ОС.
- Аттестация: программа/методика, испытания (включая облачные специфические сценарии), evidence pack, управление изменениями после аттестации.
