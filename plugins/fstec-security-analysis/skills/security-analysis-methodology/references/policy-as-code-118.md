# Policy-as-Code ориентиры (118)
- Базовые политики: IAM least privilege, шифрование at rest/in transit, публичный доступ запрещён, теги/классификация.
- Сеть: deny публичных SG/ingress, обязательные NP, egress контроль, TLS.
- Образы/CI/CD: подпись/скан, запрет неподписанных/уязвимых, контроль секретов, блокировка устаревших версий.
- Логирование/время: обязательное логирование, защита/ретеншн, синхронизация времени.
- Exception process: кто одобряет, срок исключения, алерты на нарушения.
