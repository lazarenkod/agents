## –®–∞–≥ 7: Optimization Checklist

Throughout the campaign, continuously optimize:

**Landing Page Optimization**:
- [ ] A/B test headlines (3-5 variations)
- [ ] A/B test CTAs (button copy, color, placement)
- [ ] Test form length (short vs long forms)
- [ ] Test social proof (logos vs testimonials vs metrics)
- [ ] Improve page speed (<2 seconds load time)
- [ ] Mobile optimization (50%+ traffic likely mobile)

**Email Optimization**:
- [ ] A/B test subject lines
- [ ] Test send times (day/time optimization)
- [ ] Segment lists (persona, industry, behavior)
- [ ] Personalization (dynamic content)
- [ ] Re-engagement campaigns (non-openers)

**Ad Optimization**:
- [ ] Pause underperforming ads (CPA > target)
- [ ] Scale winning ads (CPA < target, increase budget)
- [ ] Test new ad creatives (3-5 variants per channel)
- [ ] Refine targeting (exclude low-converters, focus on high-converters)
- [ ] Adjust bidding (based on conversion data)

**Content Optimization**:
- [ ] Update based on feedback (FAQs, objections)
- [ ] Repurpose top performers (blog ‚Üí video ‚Üí infographic)
- [ ] SEO optimization (target keywords, internal links)
- [ ] Distribution expansion (new channels, syndication)

**Sales Enablement**:
- [ ] Address common objections (update battlecard)
- [ ] Share customer wins (success stories, metrics)
- [ ] Refine demo script (based on what resonates)
- [ ] Lead scoring adjustment (improve MQL definition)
