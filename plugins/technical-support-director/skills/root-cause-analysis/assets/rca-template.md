# RCA Template (blameless)

## Контекст
- Инцидент/проблема, даты, impact, владельцы

## Факты/таймлайн
- Хронология с временными метками, действия/решения, алерты/сигналы

- Prevention, mitigation, detection; приоритет, владелец, срок
