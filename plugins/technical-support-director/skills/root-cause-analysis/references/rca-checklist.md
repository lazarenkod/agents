# RCA Checklist
- Факты/таймлайн полны, нет предположений.
- Причины подтверждены данными/логами, разделены на симптомы/корни.
- Есть prevention/mitigation/detection действия с владельцами/сроками.
- Алерты/мониторинг/тесты обновлены, документация/обучение указаны.
- Decision/risk log и TODO обновлены, изменения vs прошлой версии зафиксированы.
