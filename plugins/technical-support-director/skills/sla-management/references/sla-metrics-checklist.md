# Чек-лист метрик SLA
- Response/Resolution compliance по приоритетам, доступность (downtime budget)
- MTTR/MTTA/MTTD, % эскалаций, % нарушений/кредитов
- Качество коммуникаций (своевременность апдейтов), customer satisfaction по инцидентам
- Исключения/maintenance окна, корректность измерения (TZ, бизнес-часы)
- Алерты: падение compliance, всплеск P1/P2, рост кредитов/жалоб
