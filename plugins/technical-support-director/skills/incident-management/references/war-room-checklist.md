# War Room Setup Checklist

## Быстрая Активация (5 минут)

### 1. Коммуникационные Каналы
- [ ] Slack/Teams канал: #incident-[ID]
- [ ] Video conference bridge (Zoom/Meet)
- [ ] Phone bridge (backup)
- [ ] Live document (Google Docs)
- [ ] Status page готов

### 2. Роли
- [ ] Incident Commander назначен
- [ ] Technical Lead назначен
- [ ] Communications Lead назначен
- [ ] Scribe назначен
- [ ] Customer Liaison (для Enterprise incidents)

### 3. Инструменты
- [ ] Monitoring dashboards открыты
- [ ] Log aggregation доступен
- [ ] Cloud consoles готовы
- [ ] Deployment tools доступны

### 4. Stakeholders
- [ ] On-call engineer в war room
- [ ] Team lead уведомлен
- [ ] Director уведомлен (P1)
- [ ] Account managers (affected customers)

## War Room Protocols

- Join announcement: "This is [Name], [Role]"
- Mute when not speaking
- IC has final decision authority
- Updates every 15 min (P1), 30 min (P2)
- Scribe documents all actions
