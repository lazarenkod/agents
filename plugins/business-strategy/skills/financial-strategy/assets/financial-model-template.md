# Финмодель (упрощённый шаблон)

## Исходные данные
- Пользователи/клиенты, цены, скидки
- Каналы и CAC по каналам
- COGS (инфраструктура, лицензии), поддержка, продажи/маркетинг, R&D, G&A

## Сценарии
- Base / Upside / Downside (допущения по росту, ценам, оттоку)
- ARR/NRR, gross margin, payback, EBITDA

## Чувствительность
- Драйверы: цена, конверсия, churn, инфраструктурные затраты
- Таблица чувствительности и график

## Выводы
- Go/No-Go, ключевые риски, контрольные точки
