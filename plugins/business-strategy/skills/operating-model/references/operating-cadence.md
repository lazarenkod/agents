# Operating Cadence (WBR/MBR/QBR)

- **WBR:** здоровье операций/продукта, алерты, инциденты, быстрые решения.
- **MBR:** прогресс против OKR/KPI, ретроспектива инцидентов, рескопинг.
- **QBR:** стратегия/портфель, инвестиции/ресурсы, kill/double-down решения.
- Для каждого формата: цели, участники, артефакты, входные/выходные материалы, длительность, follow-up.
