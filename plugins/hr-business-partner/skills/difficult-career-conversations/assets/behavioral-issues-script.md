# Behavioral Issues Script (SBI + CLEAR)

## Structure
1. **Context**: «На планерке 12.02…»
2. **Behavior**: описать факты без оценки.
3. **Impact**: влияние на команду/клиентов/результаты.
4. **Expectation**: какие стандарты нарушены (values, policies).
5. **Reset Plan**: шаги, поддержка, последствия.

## Example
```
Context: "На демо в четверг..."
Behavior: "Ты перебил клиента трижды и закрыл вопрос без ответа."
Impact: "Это создало впечатление, что мы не слушаем требования, риск потери $3M сделки."
Expectation: "Ожидаем поведения в соответствии с ценностью 'Customer Obsession'."
Plan: "Потренируем active listening с коучем, повторная оценка через 2 недели."
```

## Notes
- Всегда предложите поддержку (коучинг, shadowing).
- Если это повторяющееся поведение — документируйте как часть PIP.
