# Career Plateau Discussion Script

## 1. Set the Tone
- Подтвердите ценность сотрудника, опишите цель беседы (поиск новых возможностей роста).

## 2. Evidence
- Конкретные наблюдения (scope не расширяется, KPI стагнируют, нет новых навыков).
- Ссылки на performance reviews, feedback, карьерные цели.

## 3. Exploration Questions
- «Какие новые вызовы вас драйвят сейчас?»
- «Что мешает переходу на следующий уровень?»
- «Какую помощь/ресурсы нужно предоставить?»

## 4. Options
- Stretch задачи, ротации, менторство, обучение, смена функции/локации.

## 5. Commitment Plan
- Конкретные шаги, KPI прогресса, сроки.
- Обновление IDP + запись в career canvas.

## 6. Follow-up
- Назначьте checkpoint (30/60/90 дней).
- Фиксируйте в markdown файл + HR записи.
