# CV Quality Scorecard

Матрица оценки качества CV/LinkedIn/Executive Bio. Заполняется после первой и третьей итерации работы с кандидатом. Каждому блоку присвойте балл от 1 до 5 и комментарий.

| Блок | Критерии | Вес | Балл | Комментарий/Факты |
| --- | --- | --- | --- | --- |
| **Стратегический фокус** | Целевая роль и seniority указаны в первой строке; есть elevator pitch (3-4 строки) | 15% |  |  |
| **Структура** | Reverse chronological порядок, ровные даты (MM/YYYY), до 2 страниц, читаемость в 90 секунд | 10% |  |  |
| **Метрики влияния** | В каждом bullet есть цифры (%, $, N, скорость), влияние на клиентов/бизнес | 20% |  |  |
| **Storytelling** | Используются формулы STAR/LOAR, глаголы действия, избегаются пассивные конструкции | 10% |  |  |
| **Технические ключевые слова** | Соответствуют job scorecard (≥ 85% keyword match для ATS) | 10% |  |  |
| **Лидерские компетенции** | Примеры управления людьми/бюджетами/roadmap, отражены масштабы | 10% |  |  |
| **Уникальность/бренд** | Четко видны суперсилы, уникальные инициативы, форумы/публикации | 5% |  |  |
| **Compliance & Accuracy** | Нет NDA-материалов, ссылки на факты проверены, даты совпадают с HRIS | 10% |  |  |
| **Форматирование** | Единые маркеры, шрифты, отступы, гиперссылки рабочие, PDF без артефактов | 5% |  |  |
| **Вложения** | CV синхронизировано с LinkedIn, promo packet, публичными биографиями | 5% |  |  |

### Пороговые значения
- **90-100%** — готово к отправке партнерам/борду.
- **80-89%** — допустимо, но требуется доработка в отмеченных блоках.
- **<80%** — нужна дополнительная итерация с `cv-quality-lab` + подключение `career-portfolio-strategy`.

### Контрольные вопросы
- Совпадают ли цифры и проекты с последним performance review?
- Есть ли примеры, демонстрирующие техническую глубину и влияние на бизнес?
- Понимает ли кандидат, как рассказать каждый bullet по модели STAR/LOAR?
