## Engineering Leadership (Staff+ IC)
| Capability | Level Expectations |
| --- | --- |
| Technical Depth | Ведёт архитектуру домена, принимает решения для нескольких команд, задает стандарты качества. |
| Influence | Ведет RFC/ADR, влияет на roadmaps соседних функций, участвует в техкомитетах. |
| Delivery | Синхронизирует 2-3 кросс-функциональных программы, управляет рисками. |
| Talent | Коучит Staff/Principal IC, строит hiring loop, удерживает ключевые таланты. |

## Product Director
| Capability | Level Expectations |
| --- | --- |
| Strategy | Формирует multi-year vision, обосновывает инвестиции, управляет портфелем >$100M ARR. |
| Customer Insight | Руководит customer councils, NPS/CSAT программами, синтезирует сигналы рынка. |
| Execution | Внедряет OKR-каскадирование, обеспечивает HIT rate релизов >80%. |
| Org Leadership | Ведёт org design, succession для PM/Design/Research, строит культуру экспериментов. |

## Go-To-Market (VP Sales/CS)
| Capability | Level Expectations |
| --- | --- |
| Revenue Engine | Прогноз точность ±3%, pipeline coverage 3x, win-rate >28% enterprise. |
| Operations | Создает RevOps/enablement механизмы, внедряет MEDDICC, QBR ритмы. |
| Partnership | Партнерства с hyperscalers/SIs, joint plays, marketing alignment. |
| Talent | План удержания AE/CSM, leadership bench, DEI цели. |

Используйте матрицы как референс при заполнении heatmap и подготовке к калибровкам.
